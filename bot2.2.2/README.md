# MES V2 — Карманная MES система (Telegram Bot)

Производственная система управления в Telegram. Полностью локальная версия с Excel-файлами.

## 🚀 Быстрый старт

### 1. Установка

```bash
# Клонировать репозиторий
cd src

# Установить зависимости
pip install -r requirements.txt
```

### 2. Получить токен бота

Напишите [@BotFather](https://t.me/BotFather) в Telegram:
- `/newbot` — создайте нового бота
- Скопируйте полученный токен

### 3. Настроить токен

Создайте файл `.env` в папке `src/`:

```
BOT_TOKEN=ваш_токен_сюда
```

Или установите переменную окружения:

```bash
export BOT_TOKEN=ваш_токен_сюда
```

### 4. Запустить бота

```bash
cd src
python bot.py
```

## 🎮 Роли и возможности

| Роль | Возможности |
|------|-------------|
| 👑 **Начальник** | Дэшборд, все партии, управление сотрудниками, брак, табель, отчёты |
| 🔧 **Технолог** | Свои задачи, проверка брака |
| ⏱ **Табельщик** | Запись часов, отчёт по часам |
| 💰 **Бухгалтер** | Загрузка ведомостей, зарплата |
| 👷 **Сотрудник** | Свои задачи, сообщить о браке, зарплата |

## 🔧 Настройка конвейера (NO-CODE)

Отредактируйте файл `data/конвейер_настройки.xlsx`:

| ID | Название этапа | Ответственный (роль) | Норма времени (мин) | Требует фотоотчёт |
|----|---------------|---------------------|-------------------|------------------|
| 1 | Раскрой ткани | закройщик | 45 | Да |
| 2 | Пошив верха | швея | 90 | Нет |
| 3 | Контроль качества | технолог | 15 | Да |
| 4 | Упаковка | упаковщик | 20 | Да |

Меняйте строки — бот подхватит новые правила при следующем запуске.

## 📋 Команды

- `/start` — Главное меню / регистрация
- `/help` — Справка
- `/newbatch` — Создать новую партию (начальник)
- `/setrole @username роль` — Назначить роль (начальник)
- `/hours @username ГГГГ-ММ-ДД часы` — Записать часы (табельщик)
- `/start_N` — Начать этап партии #N
- `/done_N` — Завершить этап партии #N
- `/defect_N_approve` — Принять дефект #N
- `/defect_N_reject` — Отклонить дефект #N

## 🔒 Конфиденциальность

- Каждый сотрудник видит только свою зарплату
- Нет общего реестра зарплат
- Брак обезличен
- Все данные хранятся локально на вашем ПК в Excel-файлах
